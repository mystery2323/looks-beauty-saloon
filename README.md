# LOOKS Beauty Salon Website

A premium beauty salon website for LOOKS Beauty Salon in DHA Multan, Pakistan.

## Features
- Responsive design with Tailwind CSS
- Bridal makeup & party makeup services
- Online booking system
- Photo gallery with lightbox
- Testimonial slider
- Mobile-friendly navigation

## Services Offered
- Bridal Makeup Packages
- Party Makeup
- Hair Couture & Treatments
- HydraFacial & Skin Care
- Nail Artistry
- Mehndi Design

## Technologies
- HTML5, Tailwind CSS, JavaScript
- Font Awesome Icons
- Google Fonts
- Google Maps API
